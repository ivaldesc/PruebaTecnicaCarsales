# Proyecto Prueba Técnica - Backend y Frontend

Este repositorio contiene dos proyectos: un **backend desarrollado en .NET 8** y un **frontend desarrollado en Angular 21**.  
A continuación se presentan todas las instrucciones para instalar, configurar y ejecutar ambos proyectos de manera local.

---

## Requisitos Previos

Antes de comenzar, asegúrate de tener instaladas las siguientes herramientas:

- [.NET 8 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)  
- [Node.js 20+](https://nodejs.org/) (incluye npm)  
- [Angular CLI 21+](https://angular.io/cli)  
- Git (opcional, para clonar el repositorio)

---

## Backend (.NET 8)

1. **Ingresar a la carpeta del backend**

```bash
cd Backend/PruebaTecnicaCarsales
dotnet restore
dotnet run
```

2. **Restaurar el proyecto**

```bash
dotnet restore
```
3. **Ejecutar el proyecto**

```bash
dotnet run
```

---
## Frontend (Angular 21)

1. **Ingresar a la carpeta del frontend**

```bash
cd Frontend
```

2. **Instalar dependencias**

```bash
npm install
```

3. **Iniciar servicio**

```bash
ng serve
```
